# Product-Lifecycle-Version-Control-Impact-Analysis-System
Web-based system that applies version control concepts to product requirements and specifications. It enables organizations to track changes made to product requirements, maintain version history, compare versions, manage approvals, perform impact analysis, and restore previous versions when necessary

# Project Name: VersionForge

Backend Stack:
- FastAPI
- MySQL
- SQLAlchemy

Features:
- Version Management
- Change Tracking
- Approval Workflow
- Version Comparison
